严格模式

变量声明

var

  • 添加var后，其所定义的变量会成为包含他的函数的局部变量，若要在函数内定义全局变量可以去掉var
  • var有声明提升的效果
  • 声明的全局变量会成为window对象的属性，var name;就会有window.name

let

  • let声明的是块作用域，是函数作用域的子集
  • let不允许在同一个块中进行多次声明
  • let不会有声明提升效果，未定义就是未定义
  • 同时let声明的全局变量不会成为window对象的属性
  • 无法用条件语句检测是否某个变量是否已经被let声明过
  • for循环中使用let可以避免变量渗透出去

const

  • 不允许修改其所赋的值
  • 不允许重复声明
  • 作用域也是块

NULL

实质上是一个空对象指针，需要保存对象时建议用NULL进行初始化

布尔类型与其他数据类型间的转换

数据类型 转换为true的值 转换为false的值
Boolean true false

Number 非零数值 0, NaN
Object 任意对象 null
Undefined N/A (不存在) undefined

Number

整数

  • 十进制可直接写出
  • 八进制需要数字的第一位为0
  • 十六进制需要数字的前面加上0x

浮点值

  • 形如a.b的形式称为浮点值
  • 也有科学计数法1e7，表示1x107
  • 精确值可达17位小数
  • 无穷用Infinity表示
  • 用isFinite()来进行判断是否为无穷

NaN

  • 不是数值，运算过程出错会出现这个值
  • 可以使用isNaN()来判断是否不是数值，假如是数值则返回false

数值转换

Number()

类型 对应结果
布尔值 true为1，false为0

null 返回0
